# Web Image Assets

- `logo_ghost_sm.png`: bundled Pirate brand mark used by app UI.
- `namespace-icons/`: bundled namespace provider logos used inside verification UI.
- `wallet-icons/`: bundled wallet, chain, and token UI icons.
- `story-protocol-logo.jpg`: bundled Story Protocol logo asset.

Images that must be served from stable root URLs, such as favicons and PWA icons, belong in `../public/`.
